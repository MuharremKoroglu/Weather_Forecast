## <h2> About This App <img src = "https://c.tenor.com/JsoERRQcZqYAAAAi/thumbs-up-joypixels.gif" width = "42"></h2>
This app is a life expectancy and shows how developers can use navigation architecture. I used the 'MaterialPageRoute' technique for navigation. As you can see, fixed styles are used in this project. That's why we put the letter 'k' at the beginning of the variables. We also created our widget. Used boxes for our widget are created with Glass Morphism concepts. Thanks to this concept we can apply some blur or opacity to a box. In the meantime, we can move from the box to the glass button. That's why Gesture Detector was used. Furthermore, the font_awesome_flutter package is used for icons. On the other hand, we used the Stack widget for the background image. In this way, we have provided improved visibility for our glass boxes. Also, some method extraction was done to improve accessibility and management. In addition to these, we used the icon button widget. In this way, we have obtained clickable buttons. As you can see inside the glass boxes in the middle, we used Slider widgets. In this way, we have created a more flexible and pleasant selection area for the user. Finally, we've made some improvements to the readability and reorganization of our code.
